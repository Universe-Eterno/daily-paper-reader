## 摘要
迭代式的工业设计-仿真优化受限于 CAD-CAE 语义鸿沟：即在多样且耦合的约束下，如何将仿真反馈转化为有效的几何编辑。为了填补这一空白，我们提出了 COSMO-Agent（闭环优化、仿真与建模编排），这是一个工具增强的强化学习（RL）框架，旨在教导大语言模型（LLM）完成闭环 CAD-CAE 过程。具体而言，我们将 CAD 生成、CAE 求解、结果解析和几何修正建模为一个交互式强化学习环境，LLM 在其中学习编排外部工具并修正参数化几何体，直到满足约束条件。为了使学习过程稳定且具备工业可用性，我们设计了一种多约束奖励机制，共同鼓励可行性、工具链鲁棒性和结构化输出的有效性。此外，我们贡献了一个符合工业标准的数据库，涵盖了 25 个组件类别及可执行的 CAD-CAE 任务，以支持真实的训练和评估。实验表明，COSMO-Agent 训练显著提升了小型开源 LLM 在约束驱动设计方面的表现，在可行性、效率和稳定性方面超过了大型开源模型和强力闭源模型。

## Abstract
Iterative industrial design-simulation optimization is bottlenecked by the CAD-CAE semantic gap: translating simulation feedback into valid geometric edits under diverse, coupled constraints. To fill this gap, we propose COSMO-Agent (Closed-loop Optimization, Simulation, and Modeling Orchestration), a tool-augmented reinforcement learning (RL) framework that teaches LLMs to complete the closed-loop CAD-CAE process. Specifically, we cast CAD generation, CAE solving, result parsing, and geometry revision as an interactive RL environment, where an LLM learns to orchestrate external tools and revise parametric geometries until constraints are satisfied. To make this learning stable and industrially usable, we design a multi-constraint reward that jointly encourages feasibility, toolchain robustness, and structured output validity. In addition, we contribute an industry-aligned dataset that covers 25 component categories with executable CAD-CAE tasks to support realistic training and evaluation. Experiments show that COSMO-Agent training substantially improves small open-source LLMs for constraint-driven design, exceeding large open-source and strong closed-source models in feasibility, efficiency, and stability.

*   **核心目标**：开发一个能够像资深工程师一样，自主编排外部工具、解析物理反馈并迭代修正几何参数的智能体（Agent），实现约束驱动的闭环工业设计优化。

### 2. 方法论
*   **核心思想**：将 CAD-CAE 迭代建模为一个**长程序列决策问题**，通过强化学习（RL）训练大语言模型（LLM）作为控制器，在包含显式失败状态的环境中进行试错学习。
*   **关键技术细节**：
    *   **MCP 工具集**：通过模型上下文协议（MCP）集成了四个工具：CAD 生成器（基于 CadQuery）、CAE 求解器（基于 FreeCAD/CalculiX）、结果提取器（解析位移和应力）和成本计算器。
    *   **强化学习算法**：采用 **GRPO（群体相对策略优化）** 算法对 Qwen3-8B 进行微调，使模型学习在多轮交互中根据数值反馈更新参数。
    *   **多约束奖励函数 ($R_{total}$)**：
        1.  **约束奖励 ($R_{cons}$)**：根据满足的物理约束（位移、应力、成本）数量给予阶梯式奖励。
        2.  **停机奖励 ($R_{stop}$)**：惩罚在满足约束后的冗余工具调用，鼓励效率。
        3.  **格式奖励 ($R_{fmt}$)**：确保输出符合可解析的 JSON 格式。
*   **工作流程**：用户输入需求 -> LLM 制定计划 -> 调用工具生成 CAD 并仿真 -> 提取物理指标 -> 判断是否满足约束 -> 若不满足则根据反馈修正参数并进入下一轮 -> 最终输出可行方案。

*   **数据集**：构建了一个工业级基准数据集，涵盖 **25 类**常见工业组件（如法兰、支架、工字梁等），包含 **20,000 个**可执行任务。
*   **场景设置**：每个任务提供初始参数、材料库、载荷条件及位移、应力和成本的硬约束阈值。

*   **小模型逆袭**：经过 COSMO-Agent 框架训练的 **8B 模型**，其全成功率（74.5%）显著超过了参数量大得多的 **Intern-S1-236B** (32.0%) 和强力闭源模型 **Gemini-3-Flash** (67.5%)。

### 7. 优点
*   **闭环鲁棒性**：显式建模了工具链失败状态，使 Agent 具备在仿真出错时自动重试或修正参数的能力。
*   **工业对齐**：贡献了一个高质量、可执行的工业组件数据集，填补了 LLM 在 CAD/CAE 领域缺乏标准 Benchmark 的空白。
*   **训练策略创新**：利用 Rollout 日志而非昂贵的二次仿真来计算奖励，大幅提升了 RL 训练的效率。

*   **物理场景单一**：目前主要集中在**线性静态分析**，尚未涉及非线性材料、流体动力学（CFD）或多物理场耦合等更复杂的工业场景。

*   **动作空间有限**：目前主要通过修改预定义的参数化模板进行优化，尚未实现完全自由形式的几何拓扑优化。
